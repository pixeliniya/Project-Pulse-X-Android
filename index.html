🩺 Project Pulse-X

Java-Based Mobile Diagnostic Screening Assistant (Built at Age 14)

🚀 The Mission

Project Pulse-X is an open-source medical AI initiative designed to run on portable mobile architectures. It provides a zero-cost, automated secondary screening layer for childhood pneumonia, optimized for frontline health workers in low-resource environments.

🧠 Technical Framework

The system utilizes computer vision image recognition trained on a verified medical chest radiograph dataset. The core Java framework handles real-time bitmap arrays to deliver instant classification output.
